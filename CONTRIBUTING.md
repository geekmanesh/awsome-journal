# Contributing is welcome

### You can contributing with suggest features, report bugs or writing codes


# 1. Suggest features

### If you want to suggest features you must send us an **issue**

## How to do it?

1. open gitlab and go to project url, then click on issues tab(on left sidebar)
2. click on new issue, and write title and description for us!
3. Done:) Thanks for sharing your beautiful ideas<3

# 2. Report a bugs

### If you see a bug in our project even it's small just tell us!

## How to do it?

Like suggesting issue it has a same way

1. Open gitlab and go to project url, then click on issue tab(on left sidebar)
2. click on new issue and write a very comprehensive explaination about that bug
3. How you encounter that bug, Do you know how to fix that?, Where exactly the bug is? All of these you must tell me
4. Thanks for reporting bugs, even small ones

# 3. Write Code For This project

### If you want to write code for this project you feel free to do it(Debugging, Adding Features)

## How to do it?

1. Fork the repository

2. Clone the project:
```sh
git clone https://gitlab.com/username/awsome-journal-api.git
```

3. Create a new branch
```sh
git checkout -b username/feature-you-add
```

4. Write your awsome codes

5. Add files and commit
```sh
# Add file
git add file1 directory1/ app/tests/test_database.py
# Commit your changes
git commit -m "feat: simple explaination of your feature"
```

6. Submit a PR
Open our gitlab repository on browser and go to **Merge Request** tab
Click on new Merge Request and submit your feature to merge
